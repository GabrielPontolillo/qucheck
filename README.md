# QuCheck

### Installation
Create a new conda environment with the following command:
```conda create -n qucheck_env python=3.11```

Activate the environment:
```conda activate qucheck_env```

Move to the qucheck directory:
```cd path/to/qucheck```

Install the requirements:
```pip install .```
